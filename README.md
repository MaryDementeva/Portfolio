#### <ins>Автоматический разбор присланных за день штрафов, с сортировкой по правилам и рассылкой на почту</ins>.  
Требуется выбрать в указанной папке штрафы, пришедшие за сегодняшний день, выбрать из них информацию, записать в файл excel и отправить на почту.
Использовался Python.
Основные этапы: 
- выборка файлов по дате создания в указанной папке (sys, os, glob, datetime);
- чтение pdf файлов и выбор из них данных (данные на автомобиль, сумма штрафа, фото номера) (PyPDF2, re);
- запись выбранных данных в эксель файл (openpyxl);
- отправка на почту (smtplib, email.message).

Разбор файлов, формирование отчетов и отправка отчета на почту в [файле](drivers/drivers.py).
