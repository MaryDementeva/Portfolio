#### <ins>Анализ эффективности рекомендательной системы через А/Б-тест с использованием статистических методов</ins>.  
Задание: Предположим, что мы построили рекомендательную систему и ожидаем, что средний чек клиента вследствие этого должен вырасти. 
Мы провели AB-тест и собрали данные по контрольной и тестовой группе. 
Необходимо предложить (с обоснованием) и провести статистический тест, который можно применить к этим данным, 
а также сделать выводы по результатам.
Синтетические данные по среднему чеку до и после включения рекомендательной системы необходимо сгенерировать самостоятельно.   

Использовался Python.  
Основные этапы:   
- генерация синтетических данных,
- анализ данных, построение графиков, расчет статистических параметров (scipy), 
- подбор критерия в зависимости от типа распределения (scipy), 
- интерпритация результатов.
  
Выполненное задание в [файле](test_projects/АВ_test.ipynb).
