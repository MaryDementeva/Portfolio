#### <ins>Реализация задачи детекции клеток крови на основе сети SSD</ins>.
Задание: Реализовать задачу детекции клеток крови на основе сети SSD (датасет BCCD).
Использовался Python.

Основные этапы:
- создан отдельный класс для формирования датасета BCCD для передачи его в модель,
- обучена модель SSD (PyTorch),
- отображены предсказанные и истинные координаты и типы клеток крови.

Скрипт выполнения задания в формате .py в [файле](training_projects/BCCD/13_1_cv1_detect.py).   
Выполненное задание с полученными результатами в формате .ipynb в [файле](training_projects/BCCD/13_1_CV1_detect.ipynb).   
Вспомогательные файлы расположены в [папке](training_projects/BCCD).
Пояснения по применению вспомогательных файлов:  
- export.py:
 файл разбоора набора данных BCCD. Проходит по всем файлам набора и формирует файл test.csv для последующей передачи данных в модель.
- test.csv:
  заранее сформированный набор характеристик клеток для последующей передачи в модель.
  Можно не запускать export.py, а сразу использовать test.csv.   
  Пояснения также даны в файле выполненного задания.
