#### <ins>Детекция посторонних/владельцев на камере видеонаблюдения </ins>.   
Задание: Обнаружение и локализация на камере видеонаблюдения постороннних лиц, исключая ложные срабатывания на животных, движение растений и владельцев дома.
Описание выложено в ветке <ins>detection</ins>.

Основные этапы (на всех этапах применялся (Pyton):

- выбор узко-специализированного датасета для дообучения модели на распознавание людей.   
  За базовый набор данных выбран датасет [thief detection dataset](https://www.kaggle.com/datasets/janstylewis7/improvedthiefdetectiondataset)  
Набор данных предназначен для обучения моделей обнаружения объектов с целью идентификации людей  и/или их подозрительного поведения на видеозаписях с камер наблюдения.  
- анализ базового датасета.    
  Скрипт выполнения в формате .py в [файле](dataset.py).
- дополнение базового датасета кадрами владельцев.
  Т.к. по условию задачи необходимо распознать попадание посторонних лиц на территорию частного дома, то датасет был дополнен кадрами владельцев.    
  Скрипт выполнения в формате .py в [файле](add_dataset.py)
- обработка, аугментация и аннотирование дополненных кадров по аналогии с базовым датасетом. Разбивка на train. test, validation наборы.    
  Скрипт выполнения в формате .py в [файле](add_dataset.py).  
- дообучение модели yolo11s на подготовленных данных.   
Скрипт выполнения в формате .py в [файле](model.py).
- валидация обученной модели на тестовых данных.   
Скрипт выполнения задания в формате .py в [файле](models_validation.py).
