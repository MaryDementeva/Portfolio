### Машинное обучение<br/>
#### <ins>Нейросетевой перенос стиля (Style Transfer)</ins>.
Задание: Разработать алгоритм переноса художественного стиля между изображениями.

Основные этапы (на всех этапах применялся (PyTorch):
- выполнена предобработка изображений в соответствии с требованими модели Vgg19,
- заданы фукции потерь content loss и style loss и матрица Грама,
- сформирована модель переноса стиля на базе предобученной сети Vgg19,
- задана функция переноса стиля,
- выполнен перенос стиля с изображения стиля на исходное изображение.
  
Выполненное задание с полученными результатами переноса разных стилей в [файле](Style_Transfer_ipynb/TransferLearn.ipynb).  
Использованные при этом изображения расположены в [папке](Style_Transfer_ipynb).   
Принятые по умолчанию изображения расположены в [папке](Image/default_imgs).

Кроме того, образ данного проекта загружен на Docker Hub. Загрузив его и запустив контейнер, можно проверить работу на изображениях, принятых по умолчанию, или на произвольных изображениях пользователя.   

<ins>Последовательность действий:</ins>  
Построение результирующего изображения на основе заданных изображений стиля:
- скачать образ    
docker pull dementevamary/transferlearning:1.0   
- создать на компьютере папку, в которую будет сохранено результирующее изображение  
  <путь к папке>  
- запустить контейнер   
docker run -v <путь к папке>:/In_Out dementevamary/transferlearning:1.0  
Выполнение скрипта займет некоторое время (порядка 5 минут)   
- результирующее изображение появится в созданной папке <путь к папке>.

Если пользователь захочет получить стилизованное изображение для своих изображений, то до запуска контейнера нужно в папку <путь к папке>
выложить изображение содержания (ОБЯЗАТЕЛЬНО назвать его  "Content.jpg") и изображение стиля (ОБЯЗАТЕЛЬНО назвать "Style.jpg"). 
Результирующее изображение также буудет сохранено в папке <путь к папке>.  

*В качестве изображений стиля лучше использовать изображения с ярко выраженным стилевым характером. В противном случае, результат может быть не очень заметен.

**Можно воспользоваться примерами изображений стиля и содержания, выложенных в папке "Examples" (ВАЖНО - перед запуском контейнера из названий убрать индексы для приведения названий к шаблонам).

Файлы образа, кроме модели vgg19, выложены в папке [папке](Style_Transfer_ipynb).
Сама модель vgg19 не выложения, т.к. размер файла превышает лимит.
