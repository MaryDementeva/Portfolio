### Машинное обучение<br/>
#### <ins>1.Детекция посторонних/владельцев на камере видеонаблюдения </ins>.   
Задание: Обнаружение и локализация на камере видеонаблюдения постороннних лиц, исключая ложные срабатывания на животных, движение растений и владельцев дома.
Описание выложено в ветке <ins>detection</ins>.

#### <ins>2. Нейросетевой перенос стиля (Style Transfer)</ins>.
Задание: Разработать алгоритм переноса художественного стиля между изображениями.   
_ПРОЕКТ ТАКЖЕ ОФОРМЛЕН КАК DOCKER ОБРАЗ_  
Описание выложено в ветке <ins>Style-Transfer</ins>.

#### <ins>3. Создание адверсариальной атаки на классификатор MNIST — генерация шума, меняющего предсказание модели</ins>.  
Задание: Обучить шум, который при добавлении к оригинальной картинке, заставит модель LeNet, обученную на распознание цифр датасета MNIST, ошибаться в пользу другого класса.   
Описание выложено в ветке <ins>Adv_noise</ins>.

#### <ins>4. Обучение модели U-Net для решения задачи семантической сегментации</ins>. 
Задание: Построить и обучить модель U-Net для семантической сегментации сгенерированных окружностей.   
Описание выложено в ветке <ins>U-Net</ins>.

#### <ins>5. Реализация задачи детекции клеток крови на основе сети SSD</ins>.
Задание: Реализовать задачу детекции клеток крови на основе сети SSD (датасет BCCD).
Описание выложено в ветке <ins>SSD_BCCD</ins>.

#### <ins>6. Построение классификатора изображений на основе предобученной нейронной сети</ins>.  
Задание: Построить бинарнай классификатор изображений. При этом значение функции потерь должно быть не более 0,3.<br/>
Описание выложено в ветке <ins>dogs_vs_cats</ins>.

#### <ins>7. Кластеризация игроков в покер в зависимости от их поведения во время игры с целью определения наилучшей стратегии</ins>.  
Задание: Разделить игроков на кластеры в зависимости от их поведения во время игры.   
Описание выложено в ветке <ins>poker</ins>.

#### <ins>8. Автоматический разбор присланных за день штрафов, с сортировкой по правилам и рассылкой на почту</ins>.   
Задание: Выбрать в указанной папке штрафы, пришедшие за сегодняшний день, выбрать из них информацию, записать в файл excel и отправить на почту.  
Описание выложено в ветке <ins>drivers</ins>.
